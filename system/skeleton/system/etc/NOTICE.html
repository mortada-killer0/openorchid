This device is powered by Linux.

Configuration Files

  • /system/etc/fonts.xml: Font configuration file.
  • /system/etc/ppp/options: PPP configuration options.
  • /system/etc/wifi/wpa_supplicant.conf: Wi-Fi network configuration file.
  • /system/etc/bluetooth/main.conf: Bluetooth configuration file.
  • /system/etc/clatd.conf: IPv6-to-IPv4 address translator configuration file.
  • /system/etc/init.d: System V init scripts directory.
  • /system/etc/nfc/libnfc.conf: NFC configuration file.
  • /system/etc/qmi-network.conf: QMI configuration file.
